# Relay Glossary

- `Controller`: application-facing coordinator that executes a domain use case.
- `Primitive`: durable concept with explicit boundaries and lifecycle.
- `Task`: tracked unit of work with state and lineage.
- `WorkPlan`: explicit plan for accomplishing an objective.
- `Artifact`: typed output connected to task provenance.
- `Event`: domain-relevant record of something that happened.
- `Workspace`: boundary that groups plans, tasks, artifacts, and events.
