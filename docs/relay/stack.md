# Relay Stack

The implementation assumes the same baseline stack used in the sampled Relay application:

- Node 22+
- pnpm
- TypeScript
- Fastify
- Vite
- React
- Vitest

Not introduced:

- Next.js
- NestJS
- Express
- workflow engines
- queue systems
- ORM layers

This keeps Sherlock inside the current StackTrack direction and avoids framework sprawl before the domain model is stable.
